This format is for `fmt.Printf(...)` in [[Golang]], never forget the `fmt`
# Formatting Verbs

| Verb      | Meaning     | Example      |
| --------- | ----------- | ------------ |
| `%b`      | Binary      | 63 -> 111111 |
| `%c`      | Character   | 63 -> ?      |
| `%d`      | Decimal     | 63 -> 63     |
| `%s`      | String      | '63' -> 63   |
| `%x`,`%X` | Hexadecimal | 63 -> 3f, 3F |

# Escape Value
`\\` -> `\`
`\t` -> Tab
`\n` -> Newline
`\"` -> `"`
